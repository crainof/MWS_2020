## Install
Webスクレイピングのために、seleniumをが必要である。

以下のコマンドでインストールする。

`pip install selenium`

さらに、seleniumの起動にはブラウザに応じたWebDriverが必要である。今回はFirefoxを使用したためhttps://github.com/mozilla/geckodriver/releases
からダウンロード
