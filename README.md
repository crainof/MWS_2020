## Install
Webスクレイピングのために、seleniumをが必要である。

以下のコマンドでインストールする。

`pip install selenium`

さらに、seleniumの起動にはブラウザに応じたWebDriverが必要である。Firefoxを使用したためhttps://github.com/mozilla/geckodriver/releases
からダウンロードし、Driverのフォルダのパスを環境変数に追加する。
